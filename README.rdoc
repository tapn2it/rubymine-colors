= Rubymine Extensions - Colors

Installation

navigate to your rubymine preferences directory (ex: cd ~/Library/Preferences/RubyMine30)
rename or delete the 'colors' directory

git clone git://github.com/tapn2it/rubymine-colors.git colors

restart RubyMine
open rubymine preferences
navigate to File -> Settings -> Editor -> Colors & Fonts
choose a color scheme from the dropdown box
click Apply
